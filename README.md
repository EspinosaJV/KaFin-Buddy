# KAFIN-BUDDY

KaFin-Buddy is my personal finance data platform that is built with python.

Essentially, the goal of this project is the automation of the collection, validation, storage, analysis, and visualization of my personal financial transactions.

## PLANNED FEATURES

- Daily transaction ingestion from Excel file
- Data validation
- ETL pipeline
- SQLite database
- Automated financial reports
- Interactive dashboard
- Monthly & Yearly analytics
- Spending insights
- Automated backups

## TECH STACK

- Python
- Pandas
- SQLite
- Streamit
- Plotly
- OpenPyXL
